# Hello World Home Assistant Add-on

A simple Hello World FastAPI add-on for Home Assistant.

## Installation

1. In Home Assistant, go to **Settings → Add-ons → Add-on Store**
2. Click the three dots (top right) → **Add repository**
3. Enter: `https://github.com/Ahmed9190/hello-world-addon`
4. Click **Add**
5. Find "Hello World" and click **Install**

## Usage

Start the add-on and access: `http://<HA_IP>:1880/`
